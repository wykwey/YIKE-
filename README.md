# 📅 课表应用

一个使用 Flutter 开发的跨平台课表工具，支持周视图、日视图和课程筛选，界面简洁，操作流畅，适用于学习和工作场景。

## ✨ 功能特色

- 周视图：查看整周课程安排
- 日视图：专注单日详细日程
- 课程筛选：灵活筛选所需课程
- 跨平台支持：兼容 Android、iOS 和桌面端

## 🚀 使用方法

1. 克隆项目
   ```bash
   git clone https://github.com/你的用户名/你的项目名.git
   cd 你的项目名
   ```
2. 安装依赖
   ```bash
   flutter pub get
   ```
3. 运行项目
   ```bash
   flutter run
   ```
## 数据结构说明：

- day: 星期几（1=周一，2=周二...7=周日）

- periods: 节数列表（支持多节连上）

- weekPattern: 周次模式，支持多种格式：

- - '1-16'表示1到16周

- - 'all'表示每周都上
